# Safe Skills

- Manifest verification for the retained package.
- JSON and YAML parse validation.
- CSV parse validation.
- XLSX openability checks with sheet-name reporting.
- WPS and duration mapping audit at the workbook/interface level.
- Product truth registry audit.
- Prompt compiler contract audit.
- Custom GPT release packaging.
- PR diff review.
- Regression checks for the retained package structure and validator outputs.

# Out Of Scope

- Rewriting hooks, body copy, CTA text, dialogue, or approved claims.
- Reclassifying claims without an explicit audit task.
- Rebuilding product truth casually.
- Changing workbook formulas or workbook content without proof and scoped approval.
