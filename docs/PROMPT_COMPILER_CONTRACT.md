# Prompt Compiler Contract

This is an operational summary of the retained BOSMAX compiler contract.

- Supported engines: GROK and Google Flow only.
- Supported source modes: Hybrid, Frames, and Ingredients.
- Final output shape: one complete 9-section prompt set per block.
- Metadata suppression: engine-facing output must not leak block plans, WPS values, debug JSON, or internal runtime labels.
- `NO_OVERLAY` is the default unless an authorized source overrides it.
- Product truth lock: product truth outranks avatar styling, scene styling, and persuasive copy.
- WPS and duration authority: WPS workbook and duration/block rules remain authority inputs and cannot be casually overridden.
